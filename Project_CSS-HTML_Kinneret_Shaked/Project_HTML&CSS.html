כדור-יד

כדוריד הוא משחק כדור קבוצתי תחרותי.
במשחק משתתפות שתי קבוצות, בכל אחת מהן שישה שחקני שדה ושוער אחד.
ולהגן על שערה מפני הקבוצה היריבה (בדומה לכדורגל).
תנועת הכדור במגרש בין השחקנים ובידי שחקן יחיד וכן זריקתו לשער,
מתבצעות באמצעות הידיים (בדומה לכדורסל).
